# Crime Analysis Dashboard (India)

## Overview

This project presents an interactive Crime Analysis Dashboard built using Python.  
The goal of the project is to transform raw crime data into meaningful insights using data analysis and visualization techniques.

The dashboard allows users to explore crime patterns across different states, years, and categories in an interactive manner.

---

## Features

- Crime distribution analysis (Donut chart)
- Year-wise trend analysis (Line chart)
- State-wise comparison (Bar chart)
- Category-wise analysis (Stacked bar chart)
- Relationship analysis between crime categories (Scatter plot)
- Growth rate analysis
- Statistical distribution (Box plot & Histogram)
- Correlation analysis (Heatmap)

- Interactive filters:
  - State selection
  - Crime category selection
  - Year range slider

---
